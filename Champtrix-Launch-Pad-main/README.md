# Champtrix Launch Pad

A comprehensive project for launching and managing Champtrix applications.

## Project Structure

- `frontend/`: React/Next.js application
- `backend/`: Firebase/Node.js backend services
- `database/`: Database schemas and Firebase rules
- `docs/`: Project documentation and planning

## Getting Started

1. Install dependencies for frontend: `cd frontend && npm install`
2. Set up Firebase project and update `.env`
3. Run the development server: `cd frontend && npm run dev`
